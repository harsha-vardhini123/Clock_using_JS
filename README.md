# Clock_using_JS


## 🚀 Getting Started

To run the clock locally:

1. Clone the repository or download the ZIP file.
2. Open `clock.html` in your browser.

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
